闲下来以后，决定把一直想搭建的博客搭建出来，
整个过程反倒比想象中要容易很多，我采用的是gihub加hexo加suka，虽然功能简单、页面简陋，但是依然很开心，毕竟很容易就踏出了第一步，希望之后多多深入，开阔眼界，学习分享更多的技术。

简单的把整个搭建过程分享一下
基本步骤
1.注册github账号
2.安装git for win
3.配置githubSSH
4.安装node
5.安装配置hexo
6.简单认识markdown
7.添加suka主题

因为很多东西之前都没有接触过，所以自然会有对新内容抵触恐惧的心理，其实勇敢迈出第一步，一切都会变的简单。

1.注册github账号没什么好说的，主要是仓库的创建，点击create new repository,repository name设置为用户名+.gihub.io,例如我的是 hxf9812.github.io，然后点击创建，完成,进入仓库，
然后创建index页面，就可以在网上打开你的页面了，例如我的：hxf9812.github.io
2.进入https://gitforwindows.org/官网下载，安装，无脑下一步
3.在git安装目录下打开git-bash.exe，输入
*ssh-keygen -t rsa -C “你的邮箱” 然后密匙生成在默认位置 c:\Users\Administrator(其实就是你的电脑用户名).ssh\id_rsa,点开查看ssh，复制粘贴到 github主页—点击头像—setting—
SSH and GPG keys下，名字可以随便起。
4.官网下载nodejs,https://nodejs.org/zh-cn/,无脑下一步安装
5.其实打开hexo官网https://hexo.io/zh-cn/index.html，已经有说明了，但是这里还是说一下，在电脑创建文件夹，用来存放hexo代码，
在cmd下，进入刚创建出来的文件夹下，npm install hexo-cli -g
